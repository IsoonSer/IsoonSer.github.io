เผยแพร่ Blog ลงบน Website ด้วย Hugo

ลองอะไรใหม่ๆ สำหรับผม คนที่ไม่ถนัดในการทำ UI ให้เว็บ

April 15, 2025

  • tag1
  • tag2

    เนื่องจากว่าผมอยากจะมีพื้นที่ที่เอาไว้สำหรับบอกเล่า แบ่งปันเรื่องราว แบ่งปันประสบการณ์มานานแล้ว แต่ก็เนื่องจากในชีวิตมีอะไรต้องทำเยอะเกิน บวกกับการผัดวันประกันพรุ่งของตัวเอง ก็เลยไม่ได้มีโอกาสได้ทำสักที มาถึงวันนี้ ผมเรียนจบ ป.ตรี แล้วครับ เย่ ก็เลยเป็นโอกาสอันดีที่จะได้เริ่มต้นลองอะไรใหม่     อะ เข้าเรื่องกันเลยดีกว่า ก็เนื่องจากว่าผมค่อนข้างที่จะไม่ enjoy กับการออกแบบ Figma ในการทำ UI ให้ Web รวมถึงไม่ค่อยชอบการเขียน CSS ผมก็เลยพยายามที่จะหาหนทางมีเว็บให้ตัวเอง แบบที่ไม่ต้องยุ่งกับสิ่งพวกนี้ โดยก่อนหน้านี้ ประมาณปีที่แล้ว ผมก็ได้ไปเห็น Web Blog ที่ชื่อว่า suksit dot com แล้วผมก็ได้ลองอ่านงานเขียนของพี่เค้าหลายอันพอสมควร แต่ก็ไม่ได้เอะใจอะไร จนกระทั่งมาวันหนึ่ง ก็เหลือบไปเห็นคำว่า Powered by Hugo บนหน้าแรกของเว็บพี่เค้า ผมก็เลยลองกดเข้าไปดู ว่ามันคืออะไร แล้วก็ได้ลองค้นหาข้อมูล ก็เลยไปพบว่า มันเป็นเครื่องมือที่ช่วยให้เราสร้าง Static Web ได้ง่าย โดยการไปนำ Template ของคนอื่นมาดัดแปลง แค่นี้เลย ผมก็เลยสนใจ ทีนี้ก็ได้ลองไปค้นหาข้อมูลใน Youtube เพื่อไปดูว่าชาวบ้านเขาใช้สิ่งนี้ทำอะไรบ้าง ก็บังเอิญไปพบเจอกับคลิปวีดีโอของคุณพี่ NetworkChuck ละผมก็ชอบดูคลิปพี่แกอยู่แล้วด้วย เพราะว่าเป็นช่องที่ทำอะไรๆ เกี่ยวกับ IT หลายๆ อย่าง ได้น่าสนใจดี ก็เลยลองนั่งดูคลิปจนจบ ก็พบว่า จุดประสงค์ของพี่แก ก็คือ แกอยากจะมีพื้นที่เอาไว้ถ่ายทอดเรื่องราวของแกเอง ให้คนอื่น ลูกหลาน ได้รับการถ่ายทอดประสบการณ์จากชีวิตของพี่แก แล้วแกก็สอนการใช้งาน Hugo ว่าจะเอามาทำเป็น Web Blog ของเราได้อย่างไร ผมก็เลยได้ลองทำตาม ลองปรับแต่งดู ก็เลยเป็นที่มาของ Blog นี้ โดย Blog นี้ ผมก็จะมาเขียนสรุปวิธีการใช้งาน Hugo สร้าง Web Blog ว่ามีขั้นตอน อะไรอย่างไร (รวมถึงจะมีแนะนำการใช้ bash script ในการช่วยอัพเดต Content จาก Obsidian ให้ไป Sync กับตัว Web Blog ที่จะถูก Publish บน Github Page ด้วย) รวมถึงแนะนำการเอา Web Blog ที่สร้างจาก Hugo นี่แหละ ไป Host บน Github Page เพื่อที่ว่าชาวโลก จะได้เข้ามาอ่านงานของเราได้ …. โอเค หลังจากที่เล่ามายืดยาว เราก็มาดูวิธีการใช้ Hugo ในการสร้าง Web Blog ของผมกันเลย

brew install hugo
